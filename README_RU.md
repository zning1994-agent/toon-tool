# Конвертер JSON-TOON

**Конвертируйте JSON в TOON — компактное кодирование без скобок, оптимизированное для LLM-промптов.**

## Что такое TOON?

TOON (Token-Oriented Object Notation) — это компактное кодирование данных JSON без скобок, разработанное для LLM-промптов. Сокращает количество токенов на ~40% по сравнению со стандартным JSON, оставаясь полностью обратимым.

- 🌐 **5 языков**: английский, упрощённый китайский, традиционный китайский, японский, французский
- 🔄 **Двусторонняя конвертация**: JSON ↔ TOON

- 📦 **Табличные массивы**: полная поддержка табличного синтаксиса TOON
- 🔍 **SEO-оптимизация**: JSON-LD, sitemap, robots.txt, Open Graph
- ⚡ **Быстрая работа**: чистый JavaScript, мгновенная конвертация

## Синтаксис TOON

### Объект
```
name: Alice
age: 30
```

### Вложенный объект
```
user:
  name: Alice
  age: 30
```

### Строковый массив (однотипные примитивы)
```
roles[2]: admin,user
```

### Табличный массив (однотипные объекты)
```
users[2]{id,name}:
  1,Alice
  2,Bob
```

### Смешанный массив
```
items[3]:
  - 1
  - name: Alice
  - [2]: x,y
```

## Быстрый пример

**JSON:**
```json
{
  "user": "Ning",
  "role": "engineer",
  "skills": ["Python", "Go"]
}
```

**TOON:**
```
user: Ning
role: engineer
skills[2]: Python,Go
```
